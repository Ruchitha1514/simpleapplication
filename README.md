# simpleapplication
This project is a simple Single Page Application (SPA) built to demonstrate my understanding of modern frontend development concepts such as component-based architecture, routing, and state management.
The focus of this assignment is not just functionality, but also code structure, usability, and clarity.

Technologies Used

Frontend Framework: React
Routing: React Router
State Management (Store): React Context API
Build Tool: Vite
Deployment Platform: CodeSandbox
